## 📖 Descrizione

Quiz interattivo a risposta multipla per prepararsi all'esame di **Informatica** del corso di Psicobiologia. Copre tutti gli argomenti principali del programma con spiegazioni dettagliate dopo ogni risposta.

Funziona direttamente nel browser, **senza installazioni** e **senza connessione internet** — perfetto da studiare su iPhone, iPad o PC.

## ✨ Funzionalità

- ✅ **Domande a risposta singola e multipla**
- 🔀 **Modalità ordine casuale** per variare lo studio
- 💡 **Spiegazione dettagliata** dopo ogni risposta
- 📊 **Punteggio finale** con valutazione e barra di progresso
- 📱 **Ottimizzato per mobile** — tasti grandi, nessun zoom indesiderato
- ⚡ **Nessuna dipendenza** — un solo file HTML, zero librerie esterne
- 🔁 **Tentativi illimitati** — ripeti il quiz quante volte vuoi

## 🚀 Come usarlo

### Sul telefono (consigliato)
1. Apri il link della demo qui sopra con **Safari** (iPhone) o **Chrome** (Android)
2. Opzionale: tocca *Condividi → Aggiungi a schermata Home* per usarlo come un'app

### In locale
1. Clona il repository:
   ```bash
   git clone https://github.com/T-Ivan2/Flashcard-quiz---Esame-informatica---Psicobiologia.git
   ```
2. Apri il file `index.html` nel browser — nessun server necessario
